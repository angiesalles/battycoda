About BattyKoda

BattyKoda is a tool for analyzing bat echolocation and social calls. It helps researchers categorize and analyze bat vocalizations.

To use BattyKoda:

  1. Navigate to your user directory (e.g., /battykoda/home/kevin/)
  2. Select or create a species folder (e.g., /battykoda/home/kevin/Efuscus/)
  3. Browse to the recorded data files
  4. Analyze and categorize calls using the interactive interface

Data Structure

BattyKoda organizes data in the following way:

/battykoda/home/              (List of user directories)
  └── username/               (User's personal directory)
     └── species_name/        (Bat species folder)
        └── project_folder/   (Project-specific data)
           └── data_files     (Bat call recordings)
