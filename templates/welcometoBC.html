About BattyCoda

BattyCoda is a tool for analyzing animal vocalizations, including but not limited to bat echolocation, bird songs, frog calls, and other animal sounds. It helps researchers categorize, classify, and analyze vocalizations from any species.

To use BattyCoda:

  1. Navigate to your user directory
  2. Select or create a species folder (e.g., Efuscus)
  3. Browse to the recorded data files
  4. Analyze and categorize calls using the interactive interface

Data Structure

BattyCoda organizes data in the following way:

/home/ (List of user directories) └── username/ (User's personal directory) └── species_name/ (Bat species folder) └── project_folder/ (Project-specific data) └── data_files (Bat call recordings)
