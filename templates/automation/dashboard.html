About Call Detection

The automated call detection system helps you:

  • Process large amounts of bat call data efficiently
  • Get probability scores for different call types
  • Automatically label tasks based on detection results
  • Track detection progress and review results

To use call detection:

  1. Create a new detection run for a task batch
  2. Monitor progress as the system analyzes your data
  3. Review detection results with probability scores
  4. Apply detection results to automatically label tasks
